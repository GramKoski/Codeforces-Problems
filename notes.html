  • Misc Notes

    Using hashmaps in c++: import map, to see if key in map: m.find(key), the rest is like dictionaries in python

    Calculating prime numbers using fast sieve of Erotesthenos algo:

    Using hashsets in c++: import unordered_set; set.insert(value) adds element, use set if you want ordered elements

    Initialize entire array with a value at every index: arr[100] = {0} -- Initializes array to all zeros

    Problem 445A was somewhat confusing DP. Takeaway is that it is only nessecary to pay attention to the cost function of recursion. Let induction do the rest and don't think about it too hard!.

    Declare a vector of certain size n: vector arr(n). Declare a vector of certain size and initialize the array with all zeros: vector arr(n,0).

    Using string datatype in c++: str + 'a' is the best way to append characters

    Using queue in c++: include queue, methods q.push but q.pop returns void so we must call q.front to get the first value then use q.pop and the rest works like any other standard collection type

    Best way to write a 2 pointer is a while loop nested within a for loop. This is a nice standard way to do it.

    Priority queue/ maxHeap in c++: include queue and declare priority_queue. queue.push(n) to add a number n. queue.top() and queue.pop() to get the largest number and delete it.

    Combine two vectors in c++ using concatenate

    Reset priority queue in c++: q = priority_queue();

    Always flush stdout with endl in interactive problems

    Often times, redeclaring something (as empty) is more efficient than clearing it

    auto data type lets the compiler "decide what data type it should be. Note: this doesn't mean an auto variable can ve reassigned to a different data type.

    Loop through element in pretty much any data type with for (auto elem: variable)

    Get substring of string with substr(start_endex, num_chars)

    Compare substrings with s1.compare(s2)

    Convert string datatype to integer using stoi()

    CF contest 991 Div 3 has a lot of number theory style problems

    CF problem Hungry Games is a cool dp problem with binary search on prefix sum array
